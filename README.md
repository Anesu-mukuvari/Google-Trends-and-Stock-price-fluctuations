# Google-Trends-and-Stock-price-fluctuations
Analytics project using Google Trends and machine learning to model lead-lag effects in JSE stock prices (FirstRand, FSR.JO). Includes code and workflow for data prep, cross-correlation, Granger causality, XGBoost modeling, results, and slides. BCom Honours, Business Analytics.
# Predicting JSE Stock Prices with Google Trends

This repository contains a BCom Honours business analytics project investigating whether Google search trends serve as leading or lagging indicators for FirstRand (FSR.JO) stock prices.

## Project Overview
- Explores alternative data (Google Trends) for stock prediction.
- Uses time series analysis, cross-correlation, Granger causality, and ML models (Random Forest, XGBoost).
- Focuses on data preprocessing, feature selection, and evaluation of predictive performance.

## Contents
- Well-commented Python code (Google Colab .py export): full analysis pipeline.
- Group presentation slides summarizing findings and insights.
- Assignment rubric and documentation.
